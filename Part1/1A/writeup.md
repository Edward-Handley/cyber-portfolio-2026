# A1   - Discover Security Concepts on campus

## Claim
I found and documented two security controls used on campus at UWA. These were;
1. NFC/Swipe card access readers
2. CCTV Surveillance 

## Control 1 - NFC reader
**where observed** Reid library and accross campus

**Security Concepts** Authentication, access control
**Threat Mitigated** The NFC access readers reduce the threat of unauthorised access to restricited areas
**How it works** The NFC reader requries a valid swipe card (issued to an authorised person) to be presented before the door is unlocked.
**Limitations / Risks** This control does have a few noteable limitations and / or risks, these are;
- Tailgating, when a unauthorised person follows an authorised person through a restricted area using there access
- If swipe cards are stolen, unauthorised access is possible until the lost / stolen cards access is revoked
